# UI-UX-Experiment4

## Aim:

To design a wireframe diagram for a generic application or website


## Algorithm:

Plan Pages – Decide the 5 pages: Welcome, Login, Products, Product Details, Checkout.

Pick Colors & Fonts – Choose 2–3 colors and readable fonts for consistency.

Design Welcome Page – Add logo, app name, tagline, and “Get Started” button.

Design Login Page – Add input fields (email, password) and login/signup buttons.

Design Product Page – Show costume images, names, prices, and categories.

Design Checkout Page – Show order summary, shipping info, payment options, and confirm button.

Review & Export – Check layout, colors, and buttons; export pages for prototype.
